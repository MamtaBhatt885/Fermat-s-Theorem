Fermat's Last Theorem Near Miss Finder
Program Description: A Python program that finds "near misses" to Fermat’s Last Theorem, where x^n + y^n = x^n is almost true. Anyone who wants to play with the theorem can enter values for n to find a range of “near misses.”

Installation: python pyinstaller You can run fermat_near_miss.exe from dlist.
